
# Stopwatch Web Application

## SkillCraft Technology Internship Task 2

This project is an interactive stopwatch web application created using HTML, CSS, and JavaScript.

## Features
- Start Stopwatch
- Pause Stopwatch
- Reset Stopwatch
- Real-time Timer Display
- Responsive Design

## Technologies Used
- HTML
- CSS
- JavaScript

## Project Description
This stopwatch application allows users to start, pause, and reset time intervals accurately. It demonstrates JavaScript timer functions, DOM manipulation, and event handling concepts.
